## 🚀 安装使用

### 1. 安装插件
将 `flashcard-achievements` 文件夹放到思源插件目录

### 2. 启用插件
在思源设置 → 插件 → 启用「闪卡成就系统」

### 3. 开始使用
- 点击顶栏 🏆 图标打开成就面板
- 正常使用 AI 闪卡插件，**无需任何额外操作**
- 成就系统会自动追踪你的行为

## 🔧 工作原理

```
【主插件 ai-flashcards-native】
        ↓ 发送 CustomEvent (只读)
【成就系统插件 (独立运行)】
        ↓ 监听事件 + 追踪行为
        ↓ 检测成就条件
        ↓ 发放奖励 + 显示鼓励
        ↓ 数据持久化保存
```

### 主插件集成（可选）

如果想让主插件发送事件，只需在关键位置添加：
```javascript
// 在生成闪卡后
document.dispatchEvent(new CustomEvent("flashcard-generated", { detail: {...} }));

// 在复习完成后
document.dispatchEvent(new CustomEvent("flashcard-reviewed", { detail: {...} }));
```

**即使不修改主插件，成就系统也能通过轮询检测正常工作！**
