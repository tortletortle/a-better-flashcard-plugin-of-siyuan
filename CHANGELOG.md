# 更新日志 / Changelog

本文件记录所有重大版本变更。格式基于 [Keep a Changelog](https://keepachangelog.com/)。

## [0.1.0] - 2026-06-13

### ✨ 新增 (Added)
- **AI 闪卡自动生成**：基于 GPT 和笔记内容自动创建多维闪卡
- **间隔重复系统**：基于 SM2 算法，智能调整复习间隔
- **知识图谱可视化**：概念关系网络和状态展示
- **训练舱沉浸模式**：全屏复习环境，减少干扰
- **闪卡库管理**：分类、搜索、导入/导出
- **停靠面板 UI**：原生思源面板集成
- **中英文本地化**：完整的双语界面支持
- **数学公式适配**：自动规范化 LaTeX 格式

### 🎨 内容 (Content)
- 内置学习示例（斜面、导数、电容）
- 间隔重复算法说明
- 多种卡片类型（概念卡、公式卡、易错卡）

### ⚡ 性能 (Performance)
- 响应式设计，支持桌面和移动
- 主题自适应，支持所有思源原生主题

### 📋 文档 (Documentation)
- 完整中英文使用指南
- AI 服务配置说明（OpenAI + 本地 LLM）
- 开发指南和贡献说明
