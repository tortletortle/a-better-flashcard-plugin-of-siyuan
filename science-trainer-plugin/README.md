# 理科训练舱（兼容入口）

这个插件已经不再承载完整训练舱逻辑。

完整功能已经迁移到 `ai-flashcards-native` 的 **训练舱** 标签页：

- 主题拆解
- AI 追问 / 改写
- 草稿确认 / 丢弃
- AI 分类
- 加入当前原生卡包
- 写入当前思源文档
- Markdown / JSON 导出

## 现在这个插件还做什么

它只保留一个兼容入口：

- 点击顶部栏按钮时，会提示功能已迁移，并跳转到 `ai-flashcards-native`
- 打开 Dock 时，也会尝试自动跳转到主插件的训练舱
- 如果主插件未启用，会给出明确提示

## 使用方式

1. 确保已经安装并启用 `ai-flashcards-native`
2. 在主插件工作台中打开 **训练舱** 标签
3. 如果你原来习惯从旧插件入口进入，也可以暂时保留这个兼容插件

## 迁移说明

- 训练舱现在与原生闪卡底层能力在同一个插件内维护
- 主插件继续暴露 `globalThis.aiFlashcardsNativeAPI`
- 旧插件不再维护独立的 AI 配置、训练状态和制卡流程

如果你已经适应新入口，可以直接停用这个兼容插件。
