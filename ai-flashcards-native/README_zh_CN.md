# AI 闪卡原生版 - 思源插件

一个强大的 AI 驱动的闪卡学习插件，为思源笔记深度集成原生停靠面板和高效的间隔重复学习系统。

## ✨ 核心功能

### 🎴 闪卡管理
- **原生间隔重复**：基于 SM2 和 Riff 算法的间隔重复系统
- **多种卡片类型**：
  - 📝 概念卡：理解核心概念
  - 📐 公式卡：记忆重要公式
  - ⚠️ 易错卡：避免常见错误
  - 🔗 关联卡：建立知识连接
  - 💡 应用卡：实际问题求解

### 🏃 训练舱模式
- 沉浸式复习体验（全屏模式）
- 实时进度追踪
- 智能卡片调度
- 学习会话统计

### 📚 闪卡库
- 按主题分类管理
- 快速搜索和筛选
- 导入/导出功能
- 卡片编辑和版本管理

### 📊 知识图谱
- 可视化学习进度
- 概念关系网络展示
- 节点状态实时更新（掌握/中等/薄弱/未学）
- 交互式图谱操作

### ⚙️ 高级特性
- 🤖 **AI 生成**：基于笔记内容自动生成高质量闪卡
- 📐 **数学公式适配**：自动规范化 LaTeX 公式格式 (`\(\)` `\[\]` → `$ $$`)
- 🔧 **智能分段修复**：保证公式在段落中正确渲染
- 🎨 **主题适配**：完全支持思源原生主题
- 🌍 **多语言**：中英文界面完全本地化

## 📦 安装

### 方式一：从思源应用市场安装（推荐）
1. 打开思源笔记
2. 进入 **设置 → 应用市场 → 插件**
3. 搜索 `AI Flashcards Native`
4. 点击**安装**

### 方式二：从源码安装（开发者）
1. 克隆仓库：
   ```bash
   git clone https://github.com/tortletortle/a-better-flashcard-plugin-of-siyuan.git
   cd a-better-flashcard-plugin-of-siyuan/ai-flashcards-native
   ```

2. 复制到思源插件目录：
   - Windows: `%AppData%\SiYuan\data\plugins\`
   - macOS: `~/Library/Application Support/SiYuan/data/plugins/`
   - Linux: `~/.config/SiYuan/data/plugins/`

3. 重启思源笔记

4. 前往 **设置 → 关于 → 插件** 启用 `AI 闪卡原生版`

## 🚀 快速开始

### 初次使用
1. 启用插件后，右侧边栏会自动出现 **AI 闪卡** 面板
2. 或通过顶部工具栏的**闪卡图标**快速打开

### 创建闪卡

#### 方法 1：AI 自动生成
1. 在**生成**选项卡中，输入或粘贴学习内容
2. 选择生成参数：
   - **主题分类**：选择卡片所属的学科（数学、物理、编程等）
   - **卡片数量**：生成 5-50 张卡片
3. 点击 **AI 生成** 按钮
4. 等待 AI 处理（通常需要配置 API 密钥）
5. 预览生成的卡片，确认无误后**保存**

#### 方法 2：手动创建
1. 切换到**编辑**选项卡
2. 填写卡片信息：
   - **问题**（正面）
   - **答案**（背面）
   - **分类标签**
   - **卡片类型**（概念/公式/易错等）
3. 点击**保存卡片**

### 复习卡片

#### 训练舱模式（推荐）
1. 在**库**选项卡选择想要复习的主题
2. 点击**开始训练**
3. 进入沉浸式全屏复习环境
4. 阅读问题，思考答案后点击**显示答案**
5. 评分选择：
   - **😤 再来一遍**：加入复习队列，明天重新考察
   - **😐 有点难**：标记为中等难度，1 周后复习
   - **😊 掌握**：间隔升级，2 周后复习
   - **🎉 太简单**：间隔升级到最大，1 个月后复习

#### 侧边栏快速复习
1. 在右侧边栏选择**复习**选项卡
2. 浏览待复习卡片
3. 点击卡片进入详情
4. 评分完成

### 查看知识图谱
1. 切换到**知识图谱**选项卡
2. 可视化显示所有概念的学习状态
3. 鼠标悬停查看卡片详情
4. 点击节点查看相关卡片列表

### 管理设置
1. 打开**设置**选项卡
2. 配置项包括：
   - AI 服务 API 密钥
   - 间隔重复参数
   - 复习提醒
   - 卡片导入/导出路径

## 🔧 AI 服务配置

### 使用 OpenAI API（推荐）

#### 第 1 步：获取 API 密钥
1. 访问 [OpenAI 平台](https://platform.openai.com/)
2. 登录或注册账户
3. 进入 **API Keys** 页面
4. 点击 **Create new secret key**
5. 复制生成的密钥（仅显示一次）

#### 第 2 步：在插件中配置
1. 打开思源 → 插件 → **AI 闪卡** → **设置**
2. 在 **AI 服务**部分：
   - 选择 **OpenAI**
   - 粘贴 API 密钥
   - 配置模型：`gpt-4` 或 `gpt-3.5-turbo`
   - 设置温度（0.3-0.7 推荐）
3. 点击**测试连接**验证配置
4. 保存设置

#### 第 3 步：开始使用
- 在**生成**选项卡输入内容
- 点击 **AI 生成**，系统会调用 OpenAI 接口

### 使用本地 LLM（离线模式）

如果你想要隐私和离线使用，可以配置本地 LLM：

1. 安装 **Ollama** 或 **LM Studio**
2. 下载本地模型（如 `Mistral` 或 `Neural Chat`）
3. 启动本地 LLM 服务（默认 localhost:11434）
4. 在插件设置中：
   - 选择 **本地 LLM**
   - 输入服务地址
   - 选择模型名称
5. 保存后即可离线使用

## 📖 详细使用指南

### 间隔重复算法说明

本插件使用改进的 SM2 算法，核心机制如下：

- **初始间隔**：首次学习 → 1 天后复习
- **动态调整**：
  - 答对 → 间隔 × 2.5
  - 答错 → 间隔重置为 1 天
  - 中等 → 间隔 × 1.2

**学习曲线示例**：
```
第 1 天: 学习新卡片
第 2 天: 复习（间隔 1 天）
第 5 天: 复习（间隔 3 天）
第 12 天: 复习（间隔 7 天）
第 31 天: 复习（间隔 19 天）
...（间隔不断增长，直至完全掌握）
```

### 卡片类型最佳实践

#### 🎯 概念卡
```
问: 什么是间隔重复？
答: 根据遗忘曲线，在最优时间点重复学习，使记忆长期保留。
```

#### 📐 公式卡
```
问: 圆的面积公式是什么？
答: A = πr²，其中 r 是圆的半径。
```

#### ⚠️ 易错卡
```
问: 为什么 √4 = ±2 这个说法错误？
答: 根号符号 √ 定义为非负平方根，所以 √4 = 2，不是 ±2。±2 是方程 x²=4 的两个解。
```

### 批量操作

#### 导入卡片
1. 准备 CSV 或 JSON 文件（详见下方格式说明）
2. 在**库**选项卡点击**导入**
3. 选择文件
4. 预览导入内容
5. 点击**确认导入**

**CSV 格式**：
```csv
问题,答案,分类,类型
什么是 Git?,分布式版本控制系统,编程,概念卡
Git 的初始化命令是什么?,git init,编程,公式卡
```

**JSON 格式**：
```json
{
  "cards": [
    {
      "front": "什么是 Git?",
      "back": "分布式版本控制系统",
      "category": "编程",
      "type": "概念卡"
    }
  ]
}
```

#### 导出卡片
1. 在**库**选项卡选择卡片集合
2. 点击**导出**
3. 选择导出格式（CSV / JSON / Anki）
4. 自动下载

### 与 Anki 互操作性

虽然本插件采用独立的间隔重复算法，但支持与 Anki 的数据交换：

1. 从 Anki 导出 `.apkg` 文件
2. 在本插件中**导入 Anki 卡片**
3. 卡片进度会重新初始化（按本插件的学习曲线）
4. 反向导出时自动转换格式

## 🔌 双插件架构说明

本项目包含两个插件：

### 1. `ai-flashcards-native`（主插件）
- **功能**：完整的 AI 闪卡系统
- **UI 类型**：原生停靠面板 + 标签页
- **推荐**：所有新用户

### 2. `science-trainer-plugin`（兼容性壳）
- **功能**：专为物理、数学等科学课程优化
- **目的**：过渡期支持，向主插件迁移
- **状态**：**已弃用**（不再维护）

#### 为什么有两个插件？

这是一个**渐进式架构迁移方案**：

| 阶段 | 时间 | 状态 |
|------|------|------|
| **Phase 1** | v0.1.0 | 两个插件并行 |
| **Phase 2** | v0.3.0 | 主插件功能完善，shell 停止更新 |
| **Phase 3** | v1.0.0 | 完全弃用 shell，迁移现有用户 |

#### 迁移指南
如果你正在使用 `science-trainer-plugin`：

1. **备份数据**：
   ```bash
   导出现有闪卡（CSV/JSON 格式）
   ```

2. **安装主插件**：
   ```bash
   按上方安装步骤安装 ai-flashcards-native
   ```

3. **导入数据**：
   ```bash
   在 ai-flashcards-native 中导入备份的卡片
   ```

4. **卸载旧插件**（可选）：
   ```bash
   从插件管理中禁用 science-trainer-plugin
   ```

## ❓ 常见问题（FAQ）

### Q1: 我的闪卡数据存储在哪里？
**A**: 卡片数据存储在思源本地数据库中，位置：
- Windows: `%AppData%\SiYuan\data\`
- macOS: `~/Library/Application Support/SiYuan/data/`
- Linux: `~/.config/SiYuan/data/`

### Q2: 如何备份我的学习进度？
**A**: 
1. 导出卡片为 JSON 或 CSV
2. 备份整个 `data` 目录
3. 使用思源内置备份功能

### Q3: AI 生成的卡片质量不好，怎么办？
**A**: 
- 调整 API 温度参数（降低会更稳定）
- 提供更详细的学习内容
- 手动编辑生成的卡片
- 尝试不同的 AI 模型

### Q4: 支持离线使用吗？
**A**: 
- 闪卡复习：✅ 完全离线支持
- AI 生成：❌ 需要配置本地 LLM（Ollama）
- 知识图谱：✅ 完全离线支持

### Q5: 可以同步到多台设备吗？
**A**: 
- 使用思源内置的同步功能（需配置云端）
- 或手动导出/导入卡片
